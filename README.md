# Marketing Campaign Performance Dashboard

## Overview
This project is an Excel-based dashboard that analyzes marketing campaign performance across platforms, regions, and audience groups. It transforms raw data into clear insights to support data-driven decision making.

## Dashboard Preview
(MarketingDashboard.png)

## Key Metrics (KPIs)
- Total Views: 31,669
- Total Subscriptions: 1,303
- Average Engagement Rate: 4.53%
- Total Ad Spend: $17,486

## Features
- Campaign performance comparison (Subscriptions by Campaign)
- Regional performance analysis (Views by Region)
- Film-level performance tracking (Views by Film)
- Audience engagement analysis (Engagement Rate by Age Group)
- Platform engagement breakdown (Likes, Shares, Comments)

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualization (Charts)
- Calculated Metrics (Engagement Rate, Conversion Rate)

## Key Insights
- Engagement rates range from ~3.7% to ~5.3%, indicating typical campaign performance
- Age group 25–34 is the most engaged and responsive audience
- Influencer campaigns generate the highest number of subscriptions
- TikTok drives the strongest engagement across platforms

## What I Learned
- Cleaning and structuring data for analysis
- Building and using pivot tables effectively
- Creating and interpreting key performance metrics
- Designing dashboards that clearly communicate insights
